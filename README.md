# ToDo List App

This is a ToDo List app where you can add your tasks, see all your tasks, delete tasks, and mark tasks as done â€” all from a simple webpage.

## Features

- Add new tasks
- View all tasks
- Delete tasks
- Mark tasks as completed

## How to Use

1. Open the app in your browser.
2. Add your tasks using the input form.
3. View your list of tasks.
4. Click to delete a task or mark it as done.

## Technologies Used

- Java (Backend)
- Spring Boot (Handles task operations)
- HTML, CSS, JavaScript (Frontend interface)

## Endpoints

- `POST /todos` - Add a new task
- `GET /todos` - View all tasks
- `DELETE /delete/{id}` - Delete a task
- `PATCH /update/{id}` - Mark task as done
